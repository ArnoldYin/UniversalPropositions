# Universal Proposition Banks

These is release 0.9 of the Universal Proposition Banks. It is built opon [release 1.4 of the Universal Dependency Treebanks](https://lindat.mff.cuni.cz/repository/xmlui/handle/11234/1-1827) and inherits their [licence](https://lindat.mff.cuni.cz/repository/xmlui/page/licence-UD-1.4).

# About

The Universal Proposition Banks are a layer of PropBank-style semantic role labeling annotation
